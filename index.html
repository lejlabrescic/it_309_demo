📝 Task Manager - FlightPHP

This is your lightweight task manager powered by FlightPHP.

🔗 How to Use the API

  • GET /tasks – Retrieve all tasks
  • POST /tasks – Create a new task
  • GET /tasks/{id} – Retrieve a task by ID
  • PUT /tasks/{id} – Update a task
  • DELETE /tasks/{id} – Delete a task
  • DELETE /tasks/{id} – Delete a task

You can test these endpoints using tools like Postman or curl. Make sure your FlightPHP server is running and your MySQL database is connected properly.
